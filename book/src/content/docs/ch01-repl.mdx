# Chapter 1: 基础 REPL + Tool Calling

本章会围绕最小可运行 agent 的执行路径展开：

1. 用户输入进入 REPL
2. 系统提示词与历史消息组装
3. 模型返回普通回复或 tool call
4. tool 执行结果回填消息历史
5. 继续循环直到生成最终回复

## 本章待补内容

- 为什么第一阶段仍保留过渡实现
