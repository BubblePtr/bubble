# Learn Agent Build

本书会带你学习如何**从零开始**构建一个通用 Agent 。

思考一个通用 Agent 需要哪些模块？以及如何构建这些模块？

由于本书的教程性质，构建时尽量不依赖三方库，但同时也会推荐优质的开源仓库供读者使用和学习。

### 从 Coding Agent 开始

为什么从 Coding Agent 开始？先明确一个指导思想。

**Coding Agent 是通用 Agent 的最佳实践，Coding Agent 是非 Coding Agent 的超集。**

通用 Agent 追求解决问题的能力，面对不同的领域依赖不同的工具，需要预定义好不同的 Function 来组合调用，而这些工具是无法穷举的。因此不如直接让大模型来写代码，代码本身就可以衍生成无数个 Function。

### prompt 即源码

本教程不会事无巨细地将源码罗列出来，但是每章节的最后会留有一份 prompt。用这些 prompt 配合 coding agent 也可以构建出一个属于你自己的、任何语言的通用 agent。
